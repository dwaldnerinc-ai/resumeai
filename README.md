# resumeai
ResumeAI — Senior Leadership Career Command Center

ResumeAI is a high-performance, single-page application (SPA) designed specifically for senior professionals in the Infrastructure, Telecom, and Project Management sectors. It strips away the "feature bloat" of traditional resume builders to focus on ATS optimization, match scoring, and regional job tracking.
🚀 Key Features
1. The Audit & Optimize Engine

    ATS Pre-Flight Check: Automatically scans for "Senior" trust signals like PMP credentials, regional licensure (Lower Mainland, BC), and metric density.

    ⚡ Auto-Optimize: One-click injection of critical keywords (PMI/PMP, Vancouver, BC) to ensure a 90+ impact score.

    Buzzword Audit: Live detection of "weak" AI-speak (e.g., leveraged, spearheaded, pragmatic) with high-impact executive alternatives.

2. High-Fidelity PDF Generator

    Universal Rule Engine: Logic-based formatting that identifies Job Headers and Bullet Points automatically—no more fighting with templates.

    Executive Aesthetic: A high-contrast, single-column layout optimized for both human recruiters and digital parsers.

    Print-to-PDF: Native browser integration for zero-loss document export.

3. Smart Job Tracker & Match Scoring

    Match Scoring: Real-time comparison of your resume against specific job descriptions to prioritize your high-probability leads.

    Lifecycle Tracking: Manage the journey from "Saved" to "Interview" with automated follow-up reminders.

    Privacy First: All data is stored locally in your browser (LocalStorage)—no resumes or API keys are ever stored on a remote server.

🛠️ Tech Stack

    Core: Pure JavaScript (Vanilla) + HTML5/CSS3.

    AI Intelligence: Powered by Claude 3.5 Sonnet (Anthropic API).

    Storage: Browser LocalStorage for persistent, private data management.

    Styling: Custom CSS with @media print optimization for perfect PDF exports.

🏁 Getting Started

    Set API Key: Click the 🔑 icon in the header and enter your Anthropic API Key.

    Import Resume: Paste your current resume text into the Resume tab.

    Optimize: Use the Audit Bar to identify gaps and click Auto-Optimize to fix them.

    Search & Track: Use the Find Jobs tab to pull regional listings and save them to your Tracker.
